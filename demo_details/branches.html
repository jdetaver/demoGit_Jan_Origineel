Branches

Brances zijn aftakkingen van de hoofdtak. Je zou ze bijvoorbeeld kunnen gebruiken om functionaliteiten in te bouwen die nog moeten getest worden. Zodra je werk af is (eventueel getest) kan je de zijtak (branch) in de hoofdtak (main-branch) inladen.

  1. Maak een branch aan (create new branch)
    1. Optie 1 : via branchnaam , linksonder
      1. klik op de branchnaam
      2. in het menu bovenaan kies je voor "+ Create new branch ..."
      3. geef een branchnaam in
      4. merk op dat je nu in de nieuwe branch werkt
    2. Optie 2 : via source control
      1. more actions
      2. Branch
      3. create branch
      4. geef een branchnaam in
      5. merk op dat je nu in de nieuwe branch werkt
    3. Optie 3 : via command line
      1. git checkout -b branchnaam
  2. Switch tussen branches
    1. Optie 1 : via branchnaam , linksonder
      1. klik op de branchnaam
      2. in het menu bovenaan kies je voor de branch die je wil
      3. merk op dat je nu in de nieuwe branch werkt
    2. Optie 2 : via command line
      1. git checkout branchnaam
  3. Branches vanop de remote binnen halen (source control->fetch)
    1. Optie 1 : via source control
      1. more actions
      2. Fetch
  4. Merge Branches
    1. Optie 1 : via source control
      1. LET OP !! activeer eerst de ontvangende branch
      2. more actions
      3. Branch
      4. Merge
      5. Kies de te mergen branch
    2. Optie 2 : via command line
      1. git checkout hooftak
      2. git merge zijtak
