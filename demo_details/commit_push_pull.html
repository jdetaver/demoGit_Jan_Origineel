Commit, push, pull (fetch & merge)

  • De GIT levenscyclus :
    1. Untracked directory : Elke map in je filesystem
    2. Working directory : een map die opgevolgd wordt door GIT (tracked) git init
    3. Staging area : files die moeten opgevolgd worden voeg je toe aan de staging area git add *
    4. Local repository : een lokale "foto" van je files in de huidige toestand git commit -m "commit bericht"
    5. Remote repository : een "foto" van je files op een andere locatie (bv : Gitbub) git push
  • Optie 1 : via "Welcome"-scherm
    1. Open VSCode
    2. In "Welcome"-scherm klik je op "Clone Git Repository"
    3. De command palette opent en vraagt naar de web URL
    4. Geef de web URL in en druk op enter
    5. Kies een folder om de repository in te bewaren
  • Optie 2 : via command palette
    1. Open de command palette
      • via menu "View"->"Command Palette"
      • de sneltoets van het command palette Ctrl+Shift+P
      • via F1
    2. typ "git"
    3. alle git commando's worden getoond
    4. kies voor "Git: Clone"
    5. Geef de web URL in en druk op enter
    6. Kies een folder om de repository in te bewaren
  • Optie 3 : via de terminal in VSCode
    1. Open een terminal via menu "Terminal"->"New terminal"
    2. Open de correcte map (bijvoorbeeld) cd "C:\xampp\htdocs\demoGIT"
    3. Clone de repository git clone https://github.com/jdetaver/demoGITYYYMM.git
