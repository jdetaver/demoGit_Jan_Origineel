  9. Branches in je project

    Scenario : je maakt een branch aan om je eigen functie in te schrijven. Ondertussen wordt de main op de remote repository aangepast op diezelfde plaats. Hoe ga je tewerk ?

    1. Update main vanuit remote repository (Pull)
    2. Update branch met info uit main (activeer branch en merge main branch into ...)
    3. Los de conflicten op als die er zijn
    4. Merge branch into main
    5. Commit lokale repository
    6. Push main naar remote
  10. OPDRACHT :
    • Update main vanuit remote repository (Pull)
    • Maak branch : branch[eigen naam]
    • Bewerk index.html door een functie toe te voegen aan het script element
    • Noem de functie roep[eigen naam] en laat deze naar de console je naam schrijven
    • Voer nu de bovenstaande stappen uit om jouw code aan de remote repository toe te voegen
  11. Ingore file : (enkel voor untracked files !!)

    • * = meerdere karakters
    • ? = 1 karakter
    • in VSC : groene files worden niet genegeerd , de grijze files wel

  12. Restore (oude versie terug zetten)
